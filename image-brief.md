# Orange Finance — AI image brief

The homepage is now wired to load these files automatically. Create an **`images`** folder next to `orange-finance-index.html`, save each file with the **exact filename** below, and they appear instantly. Until then, the page falls back to the current photo / icon tiles, so nothing ever looks broken.

## House style (paste into every prompt)
Keep all images cohesive. Append this to each prompt:

> natural documentary photography, warm golden-hour light, soft shallow depth of field, authentic and unstaged, contemporary Australian / Perth suburban setting, muted warm colour grade with subtle amber-orange tones, realistic skin and materials, shot on 35mm, high detail, no text, no logos, no watermark

Negative / avoid: `cartoon, 3D render, CGI, illustration, oversaturated, harsh flash, text overlays, distorted hands, stock-photo cheesiness`

Export: **JPG, sRGB, ~80% quality, keep each under ~300 KB** for fast load.
